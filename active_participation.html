Web sites starts here

Everything needs to start at some place. Every website consist of the following code. I call it my foundation.

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>


</body>
</html>

To learn more in depth about the code above go to w3schools.com. most of the magic happens between the <body> and the </body> text.
